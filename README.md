# Robot Simulation

### Installation
```sh
$ git clone https://github.com/irridescentrambler/robot_assignnment.git
$ cd robot_assignnment
$ bundle install
```

### Running test cases
```sh
$ rspec spec
```

### Running code 
```sh
$ ruby app/main.rb <name of input command file>
```
Ex:- ruby app/main.rb test_data.txt
